# Setup
* run `pip install -r requirements.txt`

# Config
1. create `.env`
```
USER_CISCO=your_username
PW_CISCO=your_password
```
2. edit `autoshortkey.command`
```
BASE_PATH=your_path
```


# Command
| Hotkey     |                                                    Description |
| :--------- | -------------------------------------------------------------: |
| ctrl_l + l | type username -> enter -> wait 2 sec -> type password -> enter |
| ctrl_l + q |                                                           exit |
